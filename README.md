# Calculator

This calculator is based on Python programming language and can perform the following mathematical operations.

+ Addition
+ Subtraction
+ Multiplication
+ Division
+ Exponentiation

## Core Concepts
This calculator is made for educational purpose and covers the following python concepts

+ Functions
+ Conditionals
+ For and While Loops
+ Error Handling
+ Modules
